# Sistema de colas - Backend

## Requisitos previos

    Asegúrese de que Node.js (>= 10.13.0) esté instalado en su sistema operativo.

## Tecnologías

    Se utilizó NodeJs

## Instalación

    Git clone https://github.com/Levitaxl/backend-proyecto-colas.git
    cd backend-proyecto-colas
    npm i

## Ejecutar el código

    npm run dev
